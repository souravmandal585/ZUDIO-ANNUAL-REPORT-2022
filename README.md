# ZUDIO-ANNUAL-REPORT-2022
So in this project you will see the dashboard of ZUDIO annual sales report 2022 which i created with the help of only advanced excel and nothing else.
So here i use only advanced excel and skills used that are data cleaning, data analysis, data visualization, pivot table, pivot chart, and some excel formuals.
here are the insights:-
1) COMPARE THE SALES AND ORDER IN SINGLE CHART					
2) WHICH MONTH GOT HIGHEST SALES AND ORDERS					
3) WHO PURCHASE MORE MEN OR WOMEN IN 2022					
4) WHAT ARE THE DIFFERENT ORDER STATUS IN 2022					
5) LIST TOP 10 STATES WHO CONTRIBUTES IN SALES					
6) RELEATION BETWEEN AGE AND GENDER BASED ON NUMBER OF ORDERS					
7) WHICH CHANNEL IS CONTRIBUTING MORE IN SALES					
8) HIGHEST SELLING CATEGORY, ETC..
